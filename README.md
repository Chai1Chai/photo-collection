# Photo Collection App

A React photo gallery application with category filtering, search functionality and pagination.

The app loads collection data from a local JSON file and dynamically displays photo sets.

---

## 🚀 Features

- Category filtering
- Search by collection name
- Client-side pagination
- Loading state handling
- Responsive grid layout
- Reusable collection component

---

## 🛠 Technologies Used

- React
- React Hooks (`useState`, `useEffect`)
- JavaScript (ES6)
- SCSS
- Local JSON data

---

## 🌐 Live Demo

👉 **Try it here:**  
https://Chai1Chai.github.io/photo-collection/

---

## 🔄 How It Works

- Data is fetched from a local `data.json` file
- Collections are filtered by selected category
- Search input dynamically filters results
- Pagination splits results into pages
- Loading state improves UX while fetching data
